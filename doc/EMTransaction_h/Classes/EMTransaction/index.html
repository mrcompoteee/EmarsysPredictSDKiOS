setCart:

Report the list of items in the visitor's shopping cart. 

- (void)setCart:(NSArray<EMCartItem *> *)items;
Parameters
items

Array of cart objects.

Discussion

This command should be called on every page. Make sure all cart items are passed to the command. If the visitor's cart is empty, send the empty array.

setCategory:

Report the category currently browsed by visitor. 

- (void)setCategory:(NSString *)category;
Parameters
category

Category path.

Discussion

Should be called on all category pages. Pass a valid category path.

setKeyword:

Report the keyword used by visitors to refine their searches. 

- (void)setKeyword:(NSString *)keyword;
Parameters
keyword

Keyword selected by user.

Discussion

Brands, locations, price ranges are good examples of such keywords. If your site offers such features, you can pass keywords to the recommender system with this command.

setPurchase:ofItems:

Report a purchase event. 

- (void)setPurchase:(NSString *)orderID ofItems:(NSArray<EMCartItem *> *)items;
Parameters
orderID

ID of the purchase.

items

Array of purchased objects.

Discussion

This command should be called on the order confirmation page to report successful purchases. Also make sure all purchased items are passed to the command.

setSearchTerm:

Set search terms entered by visitor. 

- (void)setSearchTerm:(NSString *)searchTerm;
Parameters
searchTerm

Search term entered by user.

Discussion

This call should be placed in the search results page.

setView:

Report a product browsed by visitor. 

- (void)setView:(NSString *)itemID;
Parameters
itemID

ID of the viewed item (consistent with the item column specified in the catalog).

Discussion

This command should be placed in all product pages – i.e. pages showing a single item in detail. Recommender features RELATED and ALSO_BOUGHT depend on this call.
